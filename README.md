# Sonar Swaggerator

SonarQube Web API OpenAPI specification (swagger) generator. 

## Requirements

* `node`

## Usage

```bash
node index.js

# > oas-generator@1.0.0 start
# > node index.js
# 
# {"openapi":"3.0.0","info":{"title":"Web API","version":"1.0"},"servers":[{"url":"http:localhost.com/api"}], "paths":{"/api/alm_integrations/check_pat":
```
